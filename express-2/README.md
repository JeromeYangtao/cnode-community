###微信登录/第三方登录
OAuth2协议

用户授权

1.三方服务打开微信客户端（网页）
2.微信获取授权结果
3.如果用户允许授权，跳转回三方服务客户端（网页）
4.三方客户端获取临时票据code，并发给三方服务器
5.服务器通过临时票据获取access_token
